# ── REQUIRED SOFTWARE (all platforms) ────────────────────────

Node.js >= 18.0.0
  macOS:   brew install node
  Windows: https://nodejs.org  (download LTS installer)
  Linux:   curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
           sudo apt install -y nodejs
  Verify:  node --version

npm >= 9.0.0
  Comes bundled with Node.js
  Verify:  npm --version

Git >= 2.30
  macOS:   brew install git
  Windows: https://git-scm.com/download/win
  Linux:   sudo apt install -y git
  Verify:  git --version

Mosquitto MQTT Broker >= 2.0
  macOS:   brew install mosquitto
  Windows: https://mosquitto.org/download  (Windows 64-bit installer)
  Linux:   sudo apt install -y mosquitto mosquitto-clients

  Must be configured with TWO listeners:
    port 1883  — raw TCP MQTT    (for ESP32 firmware)
    port 9001  — WebSocket MQTT  (for React Native app)
  Both with allow_anonymous true

  Config file locations:
    macOS:   /opt/homebrew/etc/mosquitto/mosquitto.conf
    Windows: C:\Program Files\mosquitto\mosquitto.conf
    Linux:   /etc/mosquitto/conf.d/bracelet.conf  (create this file)

  Required config content to add:
    listener 1883
    protocol mqtt

    listener 9001
    protocol websockets

    allow_anonymous true

# ── HARDWARE COMPONENTS ──────────────────────────────────────

ESP32 Dev Module (38 pins)
  Built-in WiFi 802.11 b/g/n and Bluetooth 4.2

MAX30102 Heart Rate and Pulse Oximetry Sensor
  Connection: I2C — SDA to pin 21, SCL to pin 22
  Supply: 3.3V

DS18B20 Digital Temperature Sensor (waterproof probe recommended)
  Connection: OneWire data to pin 4
  Supply: 3.3V or 5V
  Requires: 4.7k ohm pull-up resistor between data pin and VCC

MPU-6050 6-Axis IMU (Accelerometer + Gyroscope)
  Connection: I2C — SDA to pin 21, SCL to pin 22 (shared with MAX30102)
  Supply: 3.3V

NEO-6M GPS Module
  Connection: UART — TX of NEO-6M to pin 16 (RX2) of ESP32
  Supply: 3.3V
  Note: needs clear sky or window for satellite fix
  External antenna strongly recommended for indoor use

SIM800L GSM/GPRS Module
  Connection: UART — TX of SIM800L to pin 26, RX to pin 27
  Supply: 3.7V to 4.2V (CANNOT be powered from ESP32 3.3V pin)
  Requires: dedicated LiPo or 4xAA battery supply
  Requires: 100uF electrolytic capacitor across VCC and GND power pins
            to handle 2A current spikes during GSM transmission
  Requires: SIM card with active SMS plan inserted

SOS Push Button
  Connection: one leg to pin 15, other leg to GND
  Software: INPUT_PULLUP mode, triggers on falling edge

ERM Vibration Motor 3V
  Connection: positive through NPN transistor (e.g. 2N2222 or BC547)
              base of transistor to pin 2 via 1k ohm resistor
              collector to motor positive, emitter to GND
              1N4007 flyback diode across motor terminals
  Supply: 3.3V through transistor

Lithium battery 3.7V (1000mAh or larger)
  With TP4056 charging module for USB recharging

Connecting wires
Breadboard or custom PCB for assembly
